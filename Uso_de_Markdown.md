# Investigación sobre Markdown

## ¿Qué es Markdown?
Markdown es un lenguaje de marcado ligero que permite crear documentos con formato de manera sencilla.

## Ejemplo práctico
```markdown
# Ejemplo Markdown
## Listas
- Elemento A
- Elemento B

| Campo | Valor |
|-------|--------|
| Nombre | Juan Pérez |
| Rol | Estudiante |

[Ir a GitHub](https://github.com)
```

## Ventajas de usar Markdown con GitHub
- Visualización directa en repositorios.
- Control de versiones legible.
- Compatible con tablas, código y diagramas.
